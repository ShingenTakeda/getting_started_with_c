Variaveis

Tipos

C é uma linguagem tipada, onde as variaveis tem tipos que representam tipos de dados e o contexto no qual eles podem ser interpretado. os tipos são:

  • int: numeros inteiros de tamanho em X86_64 de 32 bits
  • char: characteres e numeros inteiros de tamanho de 8 bits
  • float: numeros reais de tamanho 32 bits seguindo IEEE 754
  • double: numeros reais de tamanho 64 bits seguindo IEEE 754

E estes podem ser modificados com as palavras reservadas short, long e unsigned.
Um exemplo de uso: long int.

  • short: diminui o tamanho de ints e floats pela a metade
  • long: dobra o tamanho de ints, floats e doubles
  • unsigned: deixa a representação de chars e ints para numeros apénas positivos

dependendo do compilador pode se usar apenas a palavra para representar uma int de tamanho diferente, como uma short pode ser uma inteira de 16 bits
